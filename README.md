# Водяная биржа

Играбельный Godot 4.x прототип по мотивам `Космических Рейнджеров 2`, но в сеттинге недалекого будущего после серии засух.

Игрок управляет караваном дрон-грузовиков между городами, агрокуполами, дата-центрами и полевыми лагерями. Главные ресурсы мира: вода, еда, электричество, медикаменты, вычислительные модули и люди.

## Что уже есть

- Карта районов с фракциями и опасностью маршрутов.
- Динамические цены, зависящие от воды, еды, электричества, безопасности и прогресса ИИ-комплекса.
- Торговля с грузовой вместимостью.
- Контракты на доставку.
- Дорожные события: налеты, песчаные бури, блокпосты, поборы.
- Глобальные параметры: прогресс ИИ-комплекса и напряжение экоополчения.

## Установка и запуск

### 1. Установите Godot

Проект сделан под **Godot 4.7**. Скачайте Godot с официального сайта:

https://godotengine.org/download

Подойдет обычная версия Godot 4.7 для вашей операционной системы. Дополнительные зависимости для прототипа не нужны.

### 2. Склонируйте проект

```sh
git clone git@github.com:Tomatetz/noir.git
cd noir
```

Если SSH-доступ к GitHub не настроен, можно использовать HTTPS:

```sh
git clone https://github.com/Tomatetz/noir.git
cd noir
```

### 3. Откройте проект в Godot

1. Запустите Godot.
2. Нажмите **Import**.
3. Выберите файл `project.godot` в папке проекта.
4. Откройте проект.
5. Нажмите **Run Project** или запустите сцену `res://scenes/Main.tscn`.

Проект настроен на запуск главной сцены автоматически:

```text
res://scenes/Main.tscn
```

### 4. Запуск из терминала

Если Godot доступен из терминала:

```sh
godot --path .
```

Для быстрой проверки скриптов:

```sh
godot --headless --path . --check-only --script res://scripts/MapPanel.gd
```
